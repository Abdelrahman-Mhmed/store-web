# MONOCHROME - Modern Clothing Brand Website

A complete, professional website for a modern clothing brand featuring a clean, minimalistic black and white design. Built with HTML, CSS, and JavaScript, this website provides a premium shopping experience with full e-commerce functionality.

## 🌟 Features

### Core Pages
- **Home** - Hero section, featured products, brand highlights, and newsletter signup
- **Collections** - Organized clothing categories with filtering and product display
- **About** - Brand story, mission, team information, and sustainability commitment
- **Contact** - Contact form, company information, FAQ, and social media links
- **Cart** - Shopping cart with item management and checkout preparation
- **Checkout** - Secure multi-step checkout process

### E-commerce Functionality
- Product catalog with categories (Tops, Bottoms, Dresses, Outerwear, Accessories)
- Shopping cart with localStorage persistence
- Product quick view modals with size selection
- Add to cart functionality
- Secure checkout flow
- Order summary and review
- Responsive design for all devices

### Design Features
- Clean, minimalistic black and white aesthetic
- Modern typography using Inter font family
- Smooth animations and transitions
- Professional product layouts
- Mobile-first responsive design
- Premium visual hierarchy

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate through the website to explore all features

### File Structure
```
MONOCHROME/
├── index.html              # Homepage
├── collections.html         # Collections page
├── about.html              # About page
├── contact.html            # Contact page
├── cart.html               # Shopping cart
├── checkout.html           # Checkout process

├── contact.js              # Contact page functionality

```

## 🔧 Customization

### Adding Products
To add new products, update the product data in the respective JavaScript files:
- `script.js` for homepage products
- `collections.js` for collection products
- `cart.js` for cart functionality

### Styling Changes
Modify `styles.css` to adjust:
- Colors and typography
- Layout and spacing
- Animations and transitions
- Responsive breakpoints

### Content Updates
Edit the HTML files to modify:
- Text content and copy
- Product information
- Company details
- Contact information
